# Word Vector Exploration

This repository contains a Jupyter notebook, `Word_Vector_Exploration.ipynb`, that explores how to build and analyze word vectors from text data.

The notebook walks through:
- Constructing a word co-occurrence matrix from a text corpus.
- Reducing dimensionality with Truncated SVD to obtain compact word embeddings.
- Visualizing 2D embeddings with Matplotlib to inspect semantic relationships and clusters between words.
- Comparing count-based embeddings with pre-trained GloVe vectors and analyzing properties such as analogies, polysemy, and bias in word vectors.

## File

- `Word_Vector_Exploration.ipynb` – main notebook with all implementation, experiments, and analysis.
